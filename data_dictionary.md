# Data Dictionary

## fund_master

amfi_code : Mutual fund identifier

fund_house : Fund company

scheme_name : Mutual fund name

category : Fund category

plan : Direct or Regular

---

## nav_history

date : NAV date

nav : Net Asset Value

---

## investor_transactions

investor_id : Investor ID

transaction_date : Transaction date

amount_inr : Transaction amount

transaction_type : SIP/Lumpsum

---

## scheme_performance

return_1yr_pct : 1 year return

return_3yr_pct : 3 year return

return_5yr_pct : 5 year return

expense_ratio_pct : Expense ratio
